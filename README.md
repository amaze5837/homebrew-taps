# Homebrew Tap

Personal Homebrew tap for [@amaze5837](https://github.com/amaze5837)'s projects.

## Installation

```bash
brew tap amaze5837/taps
```

```bash
brew install amaze5837/taps/cmake
brew install amaze5837/taps/cpplint
brew install amaze5837/taps/emscripten
```

## Uninstalling

```bash
brew untap amaze5837/taps
```
